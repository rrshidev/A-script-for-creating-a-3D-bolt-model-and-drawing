Генератор чертежей и 3D-моделей болтов с шестигранной головкой.

Использование:
  1. Установите зависимости: pip install -r requirements.txt
  2. Запустите скрипт: python bolt_generator.py
  3. Выберите режим:
     - 1 — пакетная генерация (диаметры 6..20, длины 20..100)
     - 2 — единичный ввод параметров
  4. Результаты сохраняются в папку ./output_bolt
     Для каждого набора создаются три файла:
       *1.dxf - чертёж
       *2.stp - 3D-модель
       *3.py  - вспомогательный скрипт с параметрами

Примечание: скрипт соответствует методическим указаниям ЕСКД (без рамки и основной надписи).
